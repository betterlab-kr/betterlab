Usage Guide

컬러 사용 가이드

헤더 / 네비게이션

배경: Primary (#124330), 텍스트: White, CTA: Secondary (#6BA88D)

CTA 버튼 / 강조

배경: Secondary (#6BA88D) 또는 Primary, hover 시 Dark 버전

히어로 섹션

배경: Hero Gradient (#124330 → #0C2E21), CTA: Accent (#DDEBC6)

하이라이트 / 배지

배경: Accent (#DDEBC6), 텍스트: Primary (#124330)

푸터

배경: Gray 900 (#0F1714), 텍스트: Gray 400 (#94A3A0)

링크 / 인터랙티브

Primary (#124330), hover 시 Secondary (#6BA88D)

CSS Variables

복사하여 사용할 수 있는 CSS 변수

:root {
  /* Primary Colors - 다크 포레스트 그린 (메인 브랜드) */
  --primary: #124330;
  --primary-light: #1A5C42;
  --primary-dark: #0C2E21;
  --primary-pale: #E8F0EC;

  /* Secondary Colors - 세이지 그린 (보조) */
  --secondary: #6BA88D;
  --secondary-light: #8FBFA8;
  --secondary-dark: #4E8A6F;
  --secondary-pale: #E5F2EB;

  /* Accent Colors - 페일 라임 그린 (강조) */
  --accent: #DDEBC6;
  --accent-light: #EAF4D9;
  --accent-dark: #C5D9A8;
  --accent-contrast: #124330;

  /* Status Colors */
  --success: #059669;
  --warning: #D97706;
  --error: #DC2626;
  --info: #0EA5E9;

  /* Neutral Colors - 그린 틴트 그레이 */
  --neutral-50: #F8FAF9;
  --neutral-100: #F1F5F3;
  --neutral-200: #E2E8E5;
  --neutral-300: #CBD5D0;
  --neutral-400: #94A3A0;
  --neutral-500: #64756F;
  --neutral-600: #475755;
  --neutral-700: #334145;
  --neutral-800: #1E2B28;
  --neutral-900: #0F1714;

  /* Gradients */
  --gradient-primary: linear-gradient(135deg, #124330 0%, #1A5C42 100%);
  --gradient-hero: linear-gradient(135deg, #124330 0%, #0C2E21 100%);
  --gradient-secondary: linear-gradient(135deg, #6BA88D 0%, #8FBFA8 100%);
  --gradient-nature: linear-gradient(135deg, #124330 0%, #6BA88D 50%, #DDEBC6 100%);
}
